# Screenshots Guide

Please save your screenshots with these exact filenames in this folder:

## Required Screenshots

1. **hero-section.png**
   - The landing page hero section
   - Shows "Your photos, everywhere you need them" with the gallery preview

2. **photo-gallery.png**
   - The main photo gallery view
   - Shows the grid of photos grouped by date (October 2025)
   - Includes the sidebar navigation

3. **folder-management.png**
   - The folders page
   - Shows folder cards (Revamp test, Hail mary, new folder 1, Atasaless)
   - Includes the "Create Folder" button

4. **login-page.png**
   - The login page
   - Shows "Welcome back" with email/password fields
   - Includes Google sign-in option

5. **register-page.png**
   - The registration page
   - Shows "Create account" with email/password/confirm password fields
   - Includes Google sign-up option

6. **photo-upload.png**
   - The photo upload modal
   - Shows "Ready to Upload" with photo preview
   - Includes "Clear All", "Add More Photos", and "Upload" buttons

## Image Specifications

- **Format**: PNG (preferred) or JPG
- **Quality**: High quality, clear and readable
- **Size**: Optimized for web (under 500KB each if possible)
- **Dimensions**: Original screenshot dimensions are fine

## After Adding Screenshots

Once you've added all screenshots, the README.md will automatically display them in the Overview section.

You can verify they're working by:
1. Opening README.md in GitHub
2. Or using a Markdown preview tool
3. Or viewing the deployed documentation
