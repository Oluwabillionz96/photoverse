## Getting Started

### Creating Your Account

1. Visit [https://photoverse-eight.vercel.app](https://photoverse-eight.vercel.app)
2. Click "Sign Up Free"
3. Enter email and password
4. Verify email with OTP code
5. Start uploading photos!

### Logging In

1. Go to login page
2. Enter email and password
3. Click "Sign in"

## Managing Photos

### Uploading Photos

1. Click "+ Add Photo" button
2. Select photos from device
3. Review preview
4. Click "Upload"

### Viewing Photos

- Grid view with all photos
- Click photo for full-screen view
- Navigate with arrow keys or swipe

### Favoriting Photos

1. Click heart icon on photo
2. View favorites in sidebar
3. Click heart again to unfavorite

## Organizing with Folders

### Creating Folders

1. Click "Folders" in navigation
2. Click "+ Create Folder"
3. Enter folder name
4. Upload photos to folder
